Линтеры
=======


Backend
-------

* `wemake-style-guideline <https://wemake-python-stylegui.de/>`_ 
* flake8

Запуск из корневой папки backend'a:

.. code-block::

    flake8

Frontend
--------

* `eslint <https://wemake-services.gitbook.io/wemake-vue-template/linting>`_ 

Запуск из корневой папки frontend'a:

.. code-block::

    npm run lint
